# QComp-scripts
## Scripts para automatizar tareas rutinarias en química computacional.

El objetivo de este repositorio es contar con un mejor seguimiento a los programas que se fueron desarrollando en el Laboratorio de Química Teórica entre 2020 y 2024.

Las primeras versiones de estos archivos están escritas en PowerShell 7, y se desarrollaron para aligerar la carga de tareas repetitivas, como buscar valores y crear nuevos archivos de entrada a partir los archivos de salida provenientes de los paquetes de estructura electrónica, en especial, Orca 5.0.3 y 5.0.4. Estos se encuentran en la carpeta _PowerShell_

Posteriormente, algunos scripts se tradujeron a Python 3, que será el lenguaje principal de este repositorio. Algunos otros se encuentran en Shell script.
