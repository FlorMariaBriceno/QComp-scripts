# QComp-scripts
Scripts para automatizar tareas rutinarias en química computacional.

*Las primeras versiones de estos archivos están escritas en PowerShell 7, y se desarrollaron para aligerar la carga de tareas repetitivas, como buscar valores dentro de los largos archivos de salida provenientes de los paquetes de estructura electrónica, en especial, Orca 5.0.3 y 5.0.4*
